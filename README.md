Веб-приложение для навигации по территории университета с использованием интерактивной карты и плавного панорамного обзора.

    🗺️ Интерактивная векторная карта с зданиями, аудиториями и точками интереса.
    🔍 Интеллектуальный поиск по объектам на карте и внутри помещений.
    🧱 Чистая архитектура на основе паттерна MVC на бэкенде и компонентного подхода на фронтенде.
    ⚡ RESTful API для эффективного взаимодействия клиентской части с сервером.
    🗄️ Надёжное хранение данных в СУБД PostgreSQL с продуманной схемой.
